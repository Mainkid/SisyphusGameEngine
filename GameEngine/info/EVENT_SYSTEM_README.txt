Вызов и обработка событий может происходить только в методах системы (унаследованной от SystemBase)
Для того, чтобы вызывать и обрабатывать событие, необходимо сделать следующее:
1. 	Объявить структуру события со данными, которые должны быть переданы из точки вызова ивента в точку обработки.
	Предпочтительно сделать это в отдельном заголовке, который вы подключите к файлам cpp систем, где происходит вызов и обработка event.
	Пример:
		struct SyTestEvent
		{
    			std::string message;
		};
2.	В точке вызова события:
		CallEvent<имя структуры, которую вы объявили>(строка-имя ивента, аргументы конструктора структуры);
	Пример:
		CallEvent<SyTestEvent>("TestEvent", "Test event was called! ");
3. 	В точке обработки события:
		Используется макрос SY_GET_THIS_FRAME_EVENT_VIEW(имя структуры, которую вы объявили) для получения view со всеми сущностями, содержащими данное событие.
		Пример:
			auto eventView = SY_GET_THIS_FRAME_EVENT_VIEW(SyTestEvent); 		//получили вью со всеми сущностями, содержащими данный ивент
			for (auto& entity : eventView)
			{		
				SyTestEvent& testEvent = eventView.get<SyTestEvent>(entity);	//получили само событие (объект) со всеми переданными в него данными
				SY_LOG_EVSY(SY_LOGLEVEL_WARNING, testEvent.message.c_str());	//логика обработки ивента
			}

Примеры содержатся в файлах Event.h и PhysicsSystem.cpp
Для логирования сообщений, связанных с системой событий создан отдельный канал EVSY. Логировать в него можно с помощью макроса SY_LOG_EVSY.
